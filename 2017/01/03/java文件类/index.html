java File 文件类使用

概述

  • File 更应该叫做一个路径
    • 文件路径或者文件夹路径
    • 路径分为绝对路径和相对路径
    • 相对路径, eclipse 一般为当前项目下

构造方法

  • File (String pathname): 根据一个路径得到 File 对象
  • File (String parent, String child) 根据一个目录和子文件/目录得到 File 对象
    • 就是把一个文件路径, 分为2个部分, 这样更灵活
  • 创建功能
    • public boolean createNewFile() 创建一个新文件, 如有有就不创建
    • public boolean mkdir() 创建文件夹, 如果存在就不创建
    • public boolean mkdirs() 创建一个文件夹, 如果父文件夹不存在, 会帮你创建出来

常用方法

  • File 重命名
    • public boolean removeTo(File dest) 把文件重命名为指的文件->虽然是改名, 但是不能传入一个名字的字符串, 只能新建一个文件,当参数传递
  • File 删除功能
    • public boolean delete() 删除文件或者文件夹
  • 重命名注意事项
    • 如果路径名相同, 就改名
    • 如果路径名不同, 就改名并剪切
  • 删除注意事项
    • 删除的文件夹, 里面不能有文件或者文件夹, 否则无法删除
    • 删除的文件不会走回收站
  • 判断功能
    • public boolean isDirectory() 判断是否是目录
    • public boolean isFile() 判断是否是文件
    • public boolean exists() 判断是否存在
    • public boolean canRead() 判断是否可读
    • public boolean canWrite() 判断是否可写
    • public boolean isHidden() 判断是否隐藏
  • 获取功能
    • public String getAbsolutePath() 获取绝对路径
    • public String getPath() 获取路径
    • public getName() 获取名称
    • public long length() 获取长度->字节数
    • public long lastModified() 获取最后一次修改的时间
    • public String[] list() 获取指定目录下的所有文件或者文件夹的名称数组
    • public File[] listFile() 获取指定目录下的所有文件或者文件夹的 File 数组
